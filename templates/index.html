Bem-vindo ao Sistema de Combate à Pobreza Menstrual

Este sistema foi desenvolvido para apoiar organizações que trabalham para combater a pobreza menstrual, fornecendo uma plataforma para gerenciar eficientemente a entrada de produtos, o controle de estoque e o monitoramento de doações.
